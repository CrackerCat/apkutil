# apkutil

## Installation

```
$ pip install git+ssh://git@github.com/aktsk/apkutil.git
```

## Usage

![debuggable](./img/debuggable.png)

### info

```
$ apkutil info sample.apk
Getting package name by aapt...
    A: package="jp.aktsk.sample" (Raw: "jp.aktsk.sample")
```

### screenshot
